## Описание
Сервис для обработки событий от хомяков и датчиков в реальном времени.
- Считает количество кругов, пробежанных каждым хомяком.
- Определяет активность хомяков.
- Отправляет уведомления при неактивности хомяков и сбоях датчиков.
- Генерирует ежедневные отчеты.
- Предоставляет REST API для получения отчетов.

### REST API
```
GET /report/daily?date=YYYY-MM-DD — получить ежедневный отчет за указанную дату.
```
